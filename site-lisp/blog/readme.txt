 blog-mode

■ blog-mode とは？

	xyzzy でブログとかの読み書きをするへなちょこモードです。
	利用条件等に関しては、LICENSE.txtをご覧下さい。

■ 動作環境

	xyzzy-0.2.2.235以降

	以下の拡張（とそれが依存する拡張）が必要です。
		- xml-rpc-modoki
		- atom-modoki

■ 対応ブログ

	▼XML-RPC
		Movable Type互換のAPIに対応
		・一応テストしてみた環境
			- ココログ
			- Movable Type 3.2日本語版 Release-2

	▼Atom API
		livedoor Blogにテスト対応。
			・利用できる機能は限られます。改行自動変換設定等は
			　Web管理画面から行ってください。
			・下書き/公開の制御ができません。保存すると公開され
			　ますので注意してください。
			・投稿にカテゴリを設定することはできますが、取得す
			　ることができません。
		はてなダイアリーにテスト対応。
			・利用できる機能は限られます。
			・下書きはサポートしていません。

■インストール

	配布アーカイブを system-root 以下に展開してください。

■設定

	・~/.xyzzy などに設定
		(autoload 'blog-mode "blog/defs" t)
		(autoload 'blog-toggle "blog/defs" t)
	※ NetInstaller経由でインストールして ni-autoload を利用している場合
	　 には必要ありません。

	・その他の設定は ~/.blog/config.l で行ってください。
		（同梱の config.l.sample 参照）

■アンインストール

	展開したファイルを削除してください。

	ブログの設定などは、~/.blog 以下に保存されていますので、忘れずに削除
	してください。

■起動

	M-x blog-mode		blog-modeを起動
	M-x blog-toggle		blog-modeを出したり引っ込めたりする

	・最初はブログが登録されていませんので、"a" で登録を行ってください。
		※ "Type" は以下のどれかを選択してください。
			- MovableType
			- Cocolog
			- livedoorBlog
			- Hiki
			- FreeStyleWiki
			- HatenaDiary

■キーバインド

	[ブログ一覧]
		Enter		ブログの投稿一覧を開く
		Space		同上
		a		ブログを登録
		p		その行のブログの設定メニューを表示
		d		その行のブログを削除
		C-k		その行のブログを切り取り
		C-y		切り取ったブログを貼り付け
		o		その行のブログをブラウザで開く
		q		終了

	[投稿一覧]
		Enter		その行の投稿を開く
		Space		同上
		n		新規投稿
		d		その行の投稿を削除
		r		投稿一覧を更新
		q		投稿一覧を閉じる

	[投稿詳細]
		右クリック	メニューを表示
		Enter		カーソル位置の項目を編集
		q		投稿詳細を閉じる
		r		投稿詳細を更新
		n		新規投稿
		o		投稿をブラウザで開く
		p		簡易プレビュー
		u		ファイルアップロード
		g		直接Ping送信
		C-x C-s		投稿をサーバーに保存
		C-c C-c		投稿をサーバーに保存して公開
		d		投稿を削除

			a	投稿日時を編集（ミニバッファ）
			A	投稿日時を現在にする
			t	タイトルを編集（ミニバッファ）
			e	本文を編集（編集画面）
			g	概要を編集（編集画面）
			k	追記を編集（編集画面）
			b	TrackBack送信先を編集（編集画面）
			c	カテゴリ選択（メニュー）

	[編集画面]
		C-x C-s		内容を保存して編集を終了
		C-c q		編集を保存せずに終了
