設定済みxyzzy

xyzzy本体のライセンスと混ぜても大丈夫なライセンスがついているLispを詰め込んでいます。

設定の方針

  • Windows風のキーバインドに設定。
    • 自分で設定ができない人を主なターゲットとしているため。
    • ただし、拡張のキーバインドまで調整しきれないので winkey.l の設定も入れる。
    • デフォルトのキーバインドの方がいい人は $XYZZY/site-lisp/siteinit.l の該当箇所を削除し、バイトコンパイルし直して、ダンプファイルを再生成すること。
  • 環境変数の設定が不要なように、すべてを $XYZZY/site-lisp/siteinit.l で設定。
    • さらにホームディレクトリを基準としたパスは、すべて xyzzy インストールディレクトリを基準としたパスに設定し直す。
    • この副次的な効果として USB メモリに入れても問題なく使えるようになっているはず。
  • ライセンス上問題のない Lisp はとりあえず詰め込む。 そのため設定せずただファイルを同梱しているだけというものもあり。
    • Lisp のバックアップとしてもある程度機能するかもしれない。
  • 設定を変更することは考えない。そのため NetInstaller は設定しない。
  • DLL や exe に依存している場合は設定しない。
